# SeismoLab

Desktop Seismology Software built with Python, ObsPy, and PySide6.

## Features

- Open MiniSEED
- Open SAC
- Waveform Viewer
- Bandpass Filter
- FFT (Coming Soon)
- Spectrogram (Coming Soon)

## Technology

- Python
- ObsPy
- PySide6
- Matplotlib
- NumPy
